# Operación contra el narcotráfico — Forense

## Contexto

Durante la investigación de un caso de narcotráfico, los agentes
intervinieron el dispositivo móvil del sospechoso. Entre los archivos
recuperados se localizó un documento llamado `piso-narco.html` que,
a simple vista, no parece contener información de utilidad.

Sin embargo, dado el contexto del caso, los investigadores sospechan
que el archivo podría haber sido modificado o renombrado para dificultar
su identificación y ocultar evidencias relevantes para la causa.

Tu tarea como analista forense es examinar el archivo, determinar su
verdadera naturaleza y recuperar la información oculta en su interior.
