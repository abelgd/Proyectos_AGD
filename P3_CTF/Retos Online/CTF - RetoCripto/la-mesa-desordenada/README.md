# La mesa desordenada — CTF Challenge

## Descripción

Este reto plantea un escenario inspirado en una investigación criminal realista.  
El participante asume el rol de un analista forense digital encargado de examinar el material incautado en un registro policial.  

Entre los elementos disponibles se incluyen una imagen y un documento de texto que podrían contener información encubierta.  
El objetivo consiste en aplicar técnicas de criptografía y OSINT para descubrir la **flag oculta** en alguno de los archivos proporcionados.

- `E-01.png` — Imagen adjunta al expediente para análisis
