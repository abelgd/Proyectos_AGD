# Caso La mesa desordenada — Investigación Criminal

## Contexto

La Guardia Civil ha abierto una investigación contra un varón de 38 años,
residente en Lebrija (Sevilla), sospechoso de estar implicado en una red
de tráfico de estupefacientes a gran escala. Tras ejecutar un registro
domiciliario el 15 de marzo de 2022, los agentes intervinieron diverso
material de interés para la causa.

Entre los efectos personales recuperados en la escena, el equipo
investigador ha considerado necesario adjuntar al expediente una imagen
que podría contener información relevante para el caso. Además, este
mismo README.md forma parte del material proporcionado y podría incluir
datos de utilidad para el análisis. Ambos documentos se proporcionan
como contexto en este directorio y quedan pendientes de análisis forense
especializado.
