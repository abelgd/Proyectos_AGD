#  Diario del atacante - CTF Challenge

## Información General
**Categoría**: Forensic  
**Dificultad**: Media  
**Puntos**: 50 pts  

## Descripción

Durante una investigación de incidente en la empresa **CORP-ES S.L.**, el equipo de respuesta confiscó el equipo portátil de un empleado sospechoso de exfiltrar información corporativa confidencial.

El análisis de red revela actividad sospechosa desde su equipo durante la **madrugada del 18 de marzo de 2024**. Se recuperó el historial del navegador Chrome del sistema comprometido.

Tu misión: analiza el historial y encuentra qué información intentó exfiltrar el atacante.
